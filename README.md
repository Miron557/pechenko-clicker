# 🍪 Печенько кликер

**Легендарный браузерный кликер, созданный с душой и печеньками.**  
Кликай, прокачивай, перерождайся, сражайся с боссами и общайся в мировом чате!

---

## 🚀 Возможности

- **👆 10 улучшений** – от бабушки до Вселенной печенья.
- **✨ Система престижа (CR)** – перерождение со всё возрастающим порогом.
- **🏆 Два мировых топа** – по печенькам и по очкам престижа (CR).
- **👥 Кланы** – создавай, вступай, прокачивай общий CR клана.
- **🥊 Клановый босс** – раз в день сражайся с гигантским печеньем вместе с кланом.
- **🐾 Питомцы** – Котопеченька, Драко-печенька и Единорог с постоянными бонусами.
- **📋 Ежедневные квесты** – получай награды за клики, покупки и перерождения.
- **📦 Battle Pass (Сезон 1)** – зарабатывай опыт, открывай уровни и эксклюзивные призы.
- **🎡 Колесо фортуны** – крути раз в 4 часа (админы без ограничений) и выигрывай призы.
- **🏷️ Магазин тегов** – покупай кастомные теги для чата за печеньки.
- **💬 Мировой чат** – общайся с игроками в реальном времени (Firebase).
- **⚡ Случайные ивенты** – Дождь из печенек, Золотое печенье, Шторм, Бабушкин бунт.
- **🎖️ Достижения** – открывай награды за успехи.
- **🏅 Ранги** – от Новичка до Бога печенек.
- **🌓 Тёмная и светлая темы** – переключай в один клик.
- **🛠️ Админ-панель** – для разработчиков, с кастомными тегами, градиентным текстом и эксклюзивными функциями.

## 🛠️ Технологии

- Чистый **HTML/CSS/JavaScript**.
- **Firebase Realtime Database** для чата и лидербордов.
- Шрифт **Montserrat** для премиум-стиля.

---

## 📦 Установка и запуск

1. Клонируй репозиторий или скачай файл `index.html`.
2. Создай проект в [Firebase](https://console.firebase.google.com), включи Realtime Database в тестовом режиме (правила `".read": true, ".write": true`).
3. Вставь свой `firebaseConfig` в код (замени плейсхолдер).
4. Открой файл через локальный сервер (Live Server, `python -m http.server 8000`) или залей на **GitHub Pages** для постоянного HTTPS-доступа.

---

## 🌐 GitHub Pages (бесплатный хостинг)

1. Переименуй файл в `index.html`.
2. Загрузи в репозиторий на GitHub.
3. Включи **Pages** в настройках репозитория (ветка `main`, корень).
4. Через минуту игра будет доступна по адресу `https://твой-ник.github.io/имя-репозитория/`.
