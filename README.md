# 🍪 Печенько кликер v1.02 BETA

Добро пожаловать в самую хрустящую браузерную кликер-игру!  
Кликай по печенью, покупай улучшения, перерождайся и общайся с друзьями в мировом чате.

---

## 🚀 Возможности

- **👆 7 улучшений** – от бабушки до квантовой печеньки.
- **✨ Система престижа** – перерождение за 1 000 000 печенек, очки престижа и магазин.
- **🏆 Мировой лидерборд** – таблица лучших игроков через Firebase (общая для всех).
- **💬 Мировой чат** – общайся с другими игроками прямо в игре (Firebase Realtime Database).
- **⚡ Случайные ивенты** – «Дождь из печенек» (x2 клик), «Золотое печенье» (x5), «Шторм» (авто-доход x2), «Бабушкин бунт» (бабушки x3).
- **🎖️ Достижения** – открывай награды за клики, покупки и ежедневные заходы.
- **📅 Ежедневная награда** – заходи каждый день и получай всё больше печенек.
- **🏅 Ранги** – от Новичка до Бога печенек.
- **🌓 Тёмная и светлая темы** – переключай по вкусу.
- **🛠️ Админ-панель** – для особых игроков (скрытая функция).

## 📦 Как развернуть у себя

1. **Скачай репозиторий** или скопируй файл `index.html`.
2. **Создай проект в [Firebase](https://console.firebase.google.com):**
   - Realtime Database в тестовом режиме (правила: `".read": true, ".write": true`).
   - Скопируй конфиг `firebaseConfig` и вставь его в код (вместо плейсхолдера).
3. **Открой через HTTPS** (Firebase требует `http://` или `https://`, не `file://`):
   - Используй локальный сервер (Live Server для VS Code, `python -m http.server 8000`).
   - Или загрузи на GitHub Pages – см. ниже.

---

## 🌐 Хостинг на GitHub Pages (бесплатно)

1. Переименуй HTML-файл в **`index.html`** и загрузи в репозиторий GitHub.
2. В настройках репозитория включи **GitHub Pages** (ветка `main`, корень).
3. Через минуту игра будет доступна по адресу `https://твой-ник.github.io/имя-репозитория/`.
